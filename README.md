# EI2

## Adicionar arquivos para o commit
Para adicionar todos os arquivos:
git add .

Para adicionar um arquivo específico:
git add nome-do-arquivo

## Criar um commit
git commit -m "mensagem do commit"

## Enviar os commits para o GitHub
git push

## Caso o repositório ainda não esteja conectado ao GitHub
git branch -M main
git remote add origin https://github.com/ArthurSchafer/EI-2-Programa-o-WEB.git
git push -u origin main
